使い方
  1. オレンジボードに音取得スケッチ（analogRead(A5) を Serial.println）を書き込み、PC と USB 接続します。
  2. ブラウザは Chrome / Edge を使い、USB接続 ボタンからポートを選択します。
  3. マイクに向かって声や手拍子を出すと、数値・メーター・グラフが反応します。
  4. RMS は一般的な「音量」に近い実効値、peak-to-peak は瞬間の振れ幅です。感度スライダーで見やすさを調整できます。
この値は相対的な音量レベルです。正確なデシベル（dB SPL）は基準音圧での校正が必要なため出していません。「大きい・小さい」を測って比べる用途に向いています。スケッチは Serial.begin(9600) 前提です。
